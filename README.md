# Eldertide Armaments (Rings and Amulets)

A comprehensive collection of 22 unique magical rings and amulets for Baldur's Gate 3, featuring powerful abilities, unique spells, and stunning visual effects.

- **13 Legendary Rings** with unique abilities and passive effects
- **9 Powerful Amulets** with distinctive magical properties

- **Journal Pages and Books** for lore and immersion

### Requirements
- Baldur's Gate 3 Patch 8 (4.1.1.6758295 or later)
- BG3 Mod Manager (Recommended)
- [BG3 Mod Fixer](https://www.nexusmods.com/baldursgate3/mods/141) (For compatibility)
- [Tutorial Chest Summoning](https://www.nexusmods.com/baldursgate3/mods/457) (Optional - for tutorial chest version)
- [BG3 Script Extender](https://github.com/Norbyte/bg3se) (Optional - enhances modding capabilities, not required for this mod)

### Installation Steps

1. **Download the Mod**
   - Download the latest version from Nexus Mods
   - Choose between Immersive (world loot) or Tutorial Chest version

2. **Using BG3 Mod Manager** (Recommended)
   - Extract the downloaded archive
   - Copy the `.pak` file to your `Baldur's Gate 3/Mods` folder
   - Open BG3 Mod Manager
   - Refresh your mod list
   - Enable "EldertideArmament"
   - Export/Save your load order

3. **Manual Installation**
   - Extract the `.pak` file
   - Place it in: `%LocalAppData%\Larian Studios\Baldur's Gate 3\Mods`
   - Edit `modsettings.lsx` to include the mod UUID: `22b848d1-5fff-4f76-a4f8-8461721e6112`

## Versions

### Immersive Version
Items are scattered throughout the game world as random loot in approximately 40 locations. Each item is **unique and drops only once per playthrough**, making them truly special finds. Items can be found in:
- Merchant inventories
- Hidden treasure chests
- Boss loot
- Special locations throughout Acts 1-3

### Tutorial Chest Version
All items are available in the tutorial chest (requires Tutorial Chest Summoning mod). Perfect for:
- Testing the mod
- Starting new playthroughs with specific builds
- Quickly accessing items after leaving the tutorial area

## Item Locations (Immersive Version)

### Consumables
Eldertide potions and elixirs can be found at various merchants and special locations:
- Sorcerous Sundries (Lower City)
- Harper Quartermaster (Moonrise Towers)
- Githyanki Quartermaster
- Various alchemist merchants throughout the game
- Special chests in hidden locations

### Journal Pages
Lore journal pages can be discovered in:
- Ketheric's personal belongings
- Hidden room magic chests
- Academy altar areas
- Various story-related chests

## Known Issues & Solutions

### Game Version Compatibility
**Problem**: Crashes or errors with BG3 Patch 8  
**Solution**:
- Update to Eldertide Armaments version 1.6.5 or later
- Version 1.6.4 and earlier are incompatible with Patch 8 (build 6758295)

### Act 2 Yurgir Chest Loot
**Problem**: Items in chest near Yurgir in Gauntlet of Shar despawn if Yurgir is convinced to kill himself before looting  
**Workaround**:
- **Always loot the chest BEFORE triggering Yurgir's dialogue**
- If already triggered, items may be lost for that playthrough
- This is a known limitation of quest-related loot in temporary locations

### Astral Champion's Ring (Ethereal Alliance)
**Issue**: Summons from Ethereal Alliance cannot be manually dismissed  
**Note**: 
- This is intentional to prevent game freezes
- Summons will automatically expire at short rest
- **Do not attempt to manually dismiss these summons** (was causing severe lag/freeze in v1.6.3)

### Items Not Appearing
**Problem**: Items don't spawn in expected locations  
**Solutions**:
- Ensure you have the latest version installed
- Check that BG3 Mod Fixer is installed and active
- Verify your load order in BG3 Mod Manager
- Some items have RNG-based spawning (check multiple vendors/locations)
- Save and reload after visiting a location

### Duplicate Items
**Problem**: Receiving multiple copies of the same item  
**Solutions**:
- This is a known limitation of the RNG loot system
- Items should only spawn once per playthrough in the immersive version
- If using tutorial chest version, multiples are expected

### Compatibility Issues
**Problem**: Mod conflicts with other equipment mods  
**Solutions**:
- Ensure BG3 Mod Fixer is active
- Check load order - equipment mods should load after base game files
- Test with minimal mod setup to identify conflicts
- Check Nexus posts for known incompatibilities

### Installation Problems
**Problem**: Mod doesn't appear in-game after installation  
**Solutions**:
- Verify `.pak` file is in the correct Mods folder
- Use BG3 Mod Manager to properly export load order
- Check that mod UUID is correctly added to `modsettings.lsx`
- Start a new game or load a save before mod was installed

### After Game Updates
**Problem**: Mod stops working after BG3 patch  
**Solutions**:
- Check Nexus Mods page for compatibility updates
- Download the latest version compatible with your game patch
- Some mods may need time to update after major patches
- Check the Posts tab on Nexus for community solutions

## Balance & Design Philosophy

This mod is designed for players who enjoy powerful, build-defining items at legendary tier:

- **Power Level**: Items are intentionally powerful, suitable for endgame content
- **Uniqueness**: Each item provides distinct gameplay options
- **Build Diversity**: Items support various character builds and playstyles
- **Visual Identity**: Custom VFX make items feel special and impactful

If you find items too powerful, consider:
- Using only 1-2 items per character
- Playing on higher difficulty settings
- Using the immersive version for natural progression
